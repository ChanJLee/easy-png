# easy-png

1. GET TINY KEY

go to [tiny-png](https://tinypng.com/developers) and fetch your api key.

2. EXPORT TINY_KEY environment variable

for example
```shell
sudo vim ~/.zshrc
export TINY_KEY=your api key
```

3. CLONE THE REPO

add fuck.sh to your environment variable

4. enjoy it

```shell
cd your project
fuck.sh -d .
```
